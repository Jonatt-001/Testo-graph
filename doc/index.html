04 / Support
Frequently Asked Questions
Is my GitHub token secure? +
Absolutely. We use OAuth 2.0 with HttpOnly, Secure cookies. Your token is never exposed to client-side JavaScript and is automatically rotated every 7 days for maximum security.
Can I use GFox with private repositories? +
Yes. GFox supports both public and private repositories. You have full, secure access to all repositories your GitHub account has permission to view.
How do I properly sign out? +
Navigate to Settings and select "Sign Out". This securely clears your authentication token and redirects you to the login page. For added security, tokens expire automatically after 7 days of inactivity.
Can I install GFox on my home screen? +
Yes. GFox is a fully functional Progressive Web App (PWA). On iOS, tap the Share button and select "Add to Home Screen". On Android, open the browser menu and select "Install app".
